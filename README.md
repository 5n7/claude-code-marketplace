# Claude Code Plugins Marketplace

A personal collection of Claude Code plugins.

## Installation

Add this marketplace to Claude Code:

```
/plugin marketplace add 5n7/claude-code-marketplace
```

Install plugins:

```
/plugin install code-review@5n7-plugins
/plugin install git-commit@5n7-plugins
```

## Plugins

### code-review

Comprehensive multi-perspective code review with 9 parallel specialized reviewers:

- Architecture
- Security
- Performance
- Bug & Correctness
- Consistency
- Testing
- Documentation
- Maintainability
- Minor Issues

### git-commit

Commit changes with well-structured Conventional Commits messages.
